# react-native-exc

New React Native program error management library

Inspired by: 
[react-native-exception-handler]([https://github.com/a7ul/react-native-exception-handler])

## Installation

```sh
npm install react-native-exc
```

```js

// ...

const result = await multiply(3, 7);
```
